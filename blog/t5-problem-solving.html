11. Problem Solving.

WHEN DID I GET BLOCKED ON A SIMPLE PROBLEM?

I got blocked on a simple problem completing 5.2, Event Handling. My code wasn't working and it turned out I had written ID instead of Id. I tried looking back through the instructions, going over each line of my code to see if there was anything which looked strange. Finally I compared my code to someone elses. And it wasn't until I went back through it individually line by line again that I picked up the mistake. I learnt that sometimes you can be really stuck on something, but it can be just a simple mistake so I shouldn't get too worried. It also highlighted the importance of taking immense care when writing the original code.

WHAT DO THESE FUNCTIONS DO?

The map() method creates a new array with the results of calling a provided function on every element in the calling array.The filter() method creates a new array with all elements that pass the test implemented by the provided function.The reduce() method applies a function against an accumulator and each element in the array (from left to right) to reduce it to a single value.
