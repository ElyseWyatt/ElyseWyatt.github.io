15. Scope.

SCOPE AND ITS PURPOSE IN JAVASCRIPT

Scope is where a variable can be accessed in a program, and there is both Global Scope and Local Scope. As you can imagine, Global Scope is when a variable can be accessed from anywhere in a program, and local scope is only available in a specific context. Variables defined in the global scope are declared outside of a block, and are available thoughout a program. It's actually best to avoid defining variables in global scope, as they can collide with variables that are locally scoped, making our code do weird things. Local Scope, or Block Scope, refers to the {} braces of a function, a loop, or an if statement, and serves as an important structural marker for our code.
