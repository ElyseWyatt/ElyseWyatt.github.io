13. JavaScript Language.

DIFFERENT JAVASCRIPT ELEMENTS AND THEIR USES

Parentheses (): Parenthesis in JavaScript are used for function calls, to surround conditional statements, or for grouping to enforce Order of Operations. Brackets []: Brackets are typically mostly used for accessing the properties of an Object (or the elements of an Array). Braces {}: Braces are used during the declaration of Object Literals, or to enclose blocks of code (function definitions, conditional blocks, loops, etc). Single Quotes '': JavaScript strings are used for storing and manipulating text, and a string can be any text inside quotes. You can use single or double quotes.
